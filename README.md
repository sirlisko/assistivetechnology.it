# Assistive Technology - Informatizzando la disabilità

Questo repository contiene risorse relative alla tesi di laurea di [Luca Lischetti](https://sirlisko.com), intitolata "Assistive Technology - Informatizzando la disabilità".

## Descrizione

La tesi, completata il 14 marzo 2010, si concentra sull'utilizzo e lo sviluppo di tecnologie assistive per migliorare la vita delle persone con disabilità. Il progetto esamina l'impatto delle soluzioni informatiche nel contesto delle disabilità, esplorando le sfide, le soluzioni innovative e le implicazioni sociali.

Questo sito web statico, costruito con Astro, presenta i contenuti educativi della tesi organizzati in capitoli dedicati alle tecnologie assistive e all'accessibilità web.

## Sviluppo

### Prerequisiti

- Node.js (versione LTS raccomandata)
- npm (package manager)

### Installazione

```bash
npm install
```

### Comandi di sviluppo

- **Avvia server di sviluppo**: `npm run dev` o `npm run start`
- **Build per produzione**: `npm run build` (include il controllo dei tipi con `astro check`)
- **Anteprima build di produzione**: `npm run preview`
- **Controllo tipi**: `npm run check`
- **Linting**: `npm run lint`
- **Formattazione**: `npm run format`

### Struttura del progetto

Il sito è organizzato seguendo la struttura della tesi:

1. **Introduzione** - Introduzione alle Tecnologie Assistive
2. **Accessibilità** - Principi di accessibilità
3. **Accessibilità Web** - WCAG, W3C, WAI e normative italiane
4. **Appendici** - Sistema Braille, classificazioni ISO, Legge Stanca

## Tecnologie utilizzate

- [Astro](https://astro.build/) - Framework per siti statici
- TypeScript - Linguaggio di programmazione
- HTML semantico per l'accessibilità
- CSS responsivo
